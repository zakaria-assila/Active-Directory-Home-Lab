# Step 01 - Environment Setup

## Goal
Prepare virtualization environment and networking for the AD lab.

## Requirements
- VirtualBox 
- Windows Server ISO (Evaluation)
- Windows 10 ISO

## Steps
1. Create VM for `DC01` (Windows Server)
2. Create VM for `CLIENT01` (Windows 10)
3. Configure networking:
   - Adapter 1: NAT
   - Adapter 2: Host-Only or Internal Network
