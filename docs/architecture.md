# Lab Architecture

## Overview
- Domain name: `lab.local`
- DC: Windows Server (AD DS + DNS)
- Client: Windows 10 (joined to domain)

## Network (Recommended)
- Adapter 1: NAT (Internet access)
- Adapter 2: Host-Only / Internal Network (Lab LAN)

## Naming
- DC hostname: `DC01`
- Client hostname: `CLIENT01`
